## 1. Windows 셋업

### 자동 설치 (권장)

```
git clone https://github.com/chadingTV/claudecode-discord.git
cd claudecode-discord
install.bat
```

자동으로 처리되는 항목:
- Node.js 설치 (winget 또는 수동 다운로드)
- Claude Code CLI 설치 (`npm install -g @anthropic-ai/claude-code`)
- npm 패키지 설치
- 프로젝트 빌드
- 바탕화면 바로가기 생성
- 트레이 앱 실행

### 사전 조건

- **Node.js 20+**: `winget install OpenJS.NodeJS.LTS` 또는 [nodejs.org](https://nodejs.org)
- **Claude Code CLI**: `npm install -g @anthropic-ai/claude-code` 후 `claude` 실행하여 로그인
- **better-sqlite3 빌드 실패 시**: `winget install Microsoft.VisualStudio.2022.BuildTools` → "Desktop development with C++" 선택

### Discord 봇 생성

1. https://discord.com/developers/applications 접속
2. **New Application** → 이름 입력 → Create
3. **Bot** 메뉴 → **Reset Token** → 토큰 복사 (`DISCORD_BOT_TOKEN`)
4. **MESSAGE CONTENT INTENT** 활성화 (필수!)
5. **OAuth2** → Scopes: `bot`, `applications.commands` 체크
6. Bot Permissions: `Send Messages`, `Embed Links`, `Read Message History`, `Use Slash Commands` 체크
7. 생성된 URL로 서버에 봇 초대

### 설정

**GUI 설정 (권장)**: `win-start.bat` 실행 → 트레이 앱이 자동으로 Settings 창 열어줌

**수동 설정**:
```
copy .env.example .env
notepad .env
```

```env
DISCORD_BOT_TOKEN=봇_토큰
DISCORD_GUILD_ID=서버_ID
ALLOWED_USER_IDS=사용자_ID
BASE_PROJECT_DIR=C:\Users\you\projects
RATE_LIMIT_PER_MINUTE=10
SHOW_COST=true
```

### 실행

```
win-start.bat          :: 백그라운드 실행 + 트레이 앱
win-start.bat --stop   :: 중지
win-start.bat --status :: 상태 확인
win-start.bat --fg     :: 포그라운드 (디버깅)
```

트레이 아이콘: 🟢 실행 중 / 🔴 중지 / 🟠 설정 필요

## 2. Linux 셋업

### 자동 설치 (권장)

```bash
git clone https://github.com/chadingTV/claudecode-discord.git
cd claudecode-discord
./install.sh
```

### 사전 조건

- **Node.js 20+**:
  ```bash
  curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
  sudo apt-get install -y nodejs
  ```
- **Claude Code CLI**:
  ```bash
  npm install -g @anthropic-ai/claude-code
  claude  # 브라우저 로그인
  ```

### 설정

```bash
cp .env.example .env
nano .env   # 또는 vim .env
```

```env
DISCORD_BOT_TOKEN=봇_토큰
DISCORD_GUILD_ID=서버_ID
ALLOWED_USER_IDS=사용자_ID
BASE_PROJECT_DIR=/home/you/projects
RATE_LIMIT_PER_MINUTE=10
SHOW_COST=true
```

### 실행

```bash
./linux-start.sh          # 백그라운드 (systemd + 트레이)
./linux-start.sh --stop   # 중지
./linux-start.sh --status # 상태 확인
./linux-start.sh --fg     # 포그라운드 (디버깅)
```

- GUI 환경: 시스템 트레이 아이콘 표시 (pystray, 자동 설치)
- 헤드리스 서버: 트레이 없이 systemd로 직접 실행
- 부팅 시 자동 시작: 트레이 메뉴에서 "시스템 시작 시 자동 실행" 체크

## 3. 안드로이드 폰에서 Discord로 사용하기

### 사전 준비

1. PC(Windows/Linux)에서 위 셋업 완료 (봇이 백그라운드에서 실행 중)
2. 안드로이드에 **Discord 앱** 설치 (Google Play Store)
3. PC에서 사용한 것과 **같은 Discord 계정**으로 로그인

### 프로젝트 등록 (최초 1회, PC에서)

Discord 채널에서:
```
/register path:my-project
```

### 안드로이드에서 사용

1. **Discord 앱** 열기
2. 봇이 초대된 **서버** 선택
3. 프로젝트가 등록된 **채널** 선택
4. **메시지 입력**: 일반 채팅처럼 Claude에게 지시 입력

   예시:
   - `src/index.ts 파일의 버그 수정해줘`
   - `새로운 API 엔드포인트 추가해줘`
   - `테스트 코드 작성해줘`

5. **실시간 진행 상황** 확인:
   - ⏳ 도구 사용 현황 + 경과 시간 표시
   - ⏹️ **Stop** 버튼으로 즉시 중단 가능

6. **도구 승인**: Claude가 파일 수정/명령 실행 요청 시 버튼 표시
   - ✅ **Approve** — 이번만 승인
   - ❌ **Deny** — 거부
   - ⚡ **Auto-approve All** — 이후 자동 승인

7. **파일 첨부**: 이미지, 문서, 코드 파일을 Discord 첨부로 전송 → Claude가 분석

8. **세션 관리**:
   - `/sessions` — 이전 세션 목록 보기, 재개/삭제
   - `/stop` — 현재 세션 중지
   - `/status` — 모든 프로젝트 상태 확인

### 푸시 알림

Discord 앱의 푸시 알림이 활성화되어 있으면:
- Claude가 승인을 요청할 때 알림
- 작업 완료 시 알림
- 폰이 잠겨 있어도 즉시 확인 가능

### 멀티 PC 허브

여러 PC에 각각 봇을 설치하면 안드로이드 한 대로 모든 PC를 제어:

```
Discord 서버
├── #회사맥-프론트엔드     ← 회사 Mac의 봇
├── #회사맥-백엔드         ← 회사 Mac의 봇
├── #집PC-사이드프로젝트    ← 집 PC의 봇
├── #클라우드서버-인프라    ← 클라우드 서버의 봇
```

채널 목록 자체가 모든 머신의 실시간 상태 대시보드가 됨.
